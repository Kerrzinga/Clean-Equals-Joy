# Clean Equals Joy

Premium cleaning services website for North Auckland — built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

## Features

- **React 18** with functional components and hooks
- **TypeScript** for type safety
- **React Router v6** for client-side routing (Home / Services / Contact pages)
- **Tailwind CSS v3** for styling
- **Responsive design** — works on mobile, tablet, and desktop
- Contact form with controlled inputs and simulated submission flow

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Pages

| Route | Description |
|---|---|
| `/` | Home page with hero and USP cards |
| `/services` | Four cleaning service offerings |
| `/contact` | Contact info and quote-request form |
